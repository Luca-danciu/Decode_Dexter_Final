# 🤖 Decode Dexter – FTC Robot Code

This repository contains the official robot code developed by **Vectron Robotics FTC Team (#17873)** for the **FIRST Tech Challenge (FTC)** season.

---

## 🚀 About Us

We are **Vectron Robotics**, a robotics team based in Dej, Romania, participating in the FIRST Tech Challenge — an international STEM competition where students design, build, and program robots to compete in a yearly challenge.

Our team focuses on:

* 💡 Innovation & engineering
* 🤝 Teamwork & collaboration
* 🧠 Problem-solving through software & hardware
* 🏆 Competitive performance in robotics events

We have participated in multiple competitions and achieved strong results, including winning alliances and top placements in national scrimmages. ([Radio Fir][1])

---

## 🧠 About the Code

This repository contains the full control system for our robot, including:

### 🔹 Core Features

* Autonomous routines (pre-programmed paths and actions)
* TeleOp control (driver-controlled period)
* Subsystem-based architecture (modular design)
* Hardware abstraction layer for easier testing and reuse
* Sensor integration (motors, encoders, IMU, etc.)

### 🔹 Technologies Used

* Java (FTC SDK)
* Android Studio environment
* FTC Control System

---

## ⚙️ Robot Capabilities

Our robot is designed to:

* Execute precise autonomous movements
* Perform efficient scoring mechanisms
* Provide reliable driver control during matches
* Adapt strategy dynamically during gameplay

Each subsystem is optimized for performance, reliability, and fast iteration during competitions.

## 📂 Project Structure

```bash
TeamCode/
 ├── drive/          # Movement and drivetrain logic
 ├── opmode/         # Autonomous & TeleOp modes
 ├── subsystems/     # Robot subsystems (intake, lift, etc.)
 ├── util/           # Utility classes and helpers
```

## 🛠️ Setup & Usage

1. Clone the repository:

```bash
git clone https://github.com/Luca-danciu/Decode_Dexter_Final.git
```

2. Open in Android Studio

3. Connect to FTC Robot Controller

4. Build & Run the code on the robot
